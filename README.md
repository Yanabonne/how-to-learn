# Научиться учиться
## Welcome to my first project :)

Этот учебный проект расскажет вам о том, как научиться учиться:
* Техники обучения Барбары Оакли
* Всё о прокрастинации
* Метод Фейнмана
* Принципы обучения Джоша Кауфмана
